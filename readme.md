### How to contribute

-   Fork repo
-   Create new branch
-   Add your profile to `contributors.json`. (See format below)
-   Submit pull request

#### Profile structure

```json copy
{
    "name": "",
    "bio": "",
    "socials": [
        {
            "platform": "twitter",
            "username": ""
        },
        {
            "platform": "linkedin",
            "username": ""
        },
        {
            "platform": "github",
            "username": ""
        }
    ]
}
```
